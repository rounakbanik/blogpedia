About

Blogpedia is a lightweight blog engine built in Python on the Google App Engine using the webapp2 framework.

Features

  • Free. Users can publish an unlimited number of posts on their blog
  • Posts can be created, edited and deleted
  • Users can publish text, images, videos and links
  • Registered users can comment on each other's blogs
  • Users also have access to their entire archive of posts, sorted yearwise
  • Posts can be liked and shared on Facebook, Twitter and Google+
  • Blogpedia also features an API to communicate with other websites in JSON
